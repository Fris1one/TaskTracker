# 📋 TaskTracker

TaskTracker — это десктопное приложение для управления задачами с календарём и системой приоритетов. Написано на Go с использованием фреймворка [Fyne](https://fyne.io/).

## ✨ Возможности

- 📅 **Календарь** — просмотр задач по дням, переключение месяцев и годов
- ⭐ **Избранное** — отмечайте важные задачи звёздочкой
- 📊 **Приоритеты** — 5 уровней приоритета: от Lowest до Highest
- 🔄 **Стадии выполнения** — 3 стадии: "Started" → "In progress" → "Completed"
- 💾 **Автосохранение** — данные сохраняются в папку пользователя
- 🖼️ **Встроенные ресурсы** — все картинки внутри исполняемого файла

### Требования

- Go 1.21+
- Fyne CLI
